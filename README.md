# Projeto Microserviços: Cliente & Pedido 
**Matéria:** Desenvolvimento Web 3 - IFSP

Este repositório contém uma aplicação baseada em microserviços desenvolvida como tarefa prática para a disciplina de Desenvolvimento Web 3. O projeto foca na criação de APIs RESTful integradas, utilizando Node.js e comunicação assíncrona.

---

## Tecnologias Utilizadas
* **Node.js** (Ambiente de execução)
* **Express** (Framework web)
* **Sequelize** (ORM para conexão com banco de dados)
* **MySQL** (Banco de Dados Relacional)
* **Node-fetch** (Comunicação entre serviços)
* **Dotenv** (Gerenciamento de variáveis de ambiente)
